WS Collector is in beta

This software is not quite ready for Google Play yet, so I'm distribuiting it directly in a effort to get as much feedback as I can to improve the app. If you're willing to install it, please press the Download button in your phone.

If you don't feel confortable installing Android apps from outside Google Play, you shouldn't. Wait for the app to be available in the Play Store after the beta has ended.

Befor you Install

To be able to open the .APK, you must allow your phone to install apps from unknown sources. This will allow you to get software that is not distributed via Google Play Store.

  1. Go to Settings > Security (or Applications, in Android 2.3)
  2. Check the box "Unknown Sources"

Frequently Asked Questions

  • Q1: I downloaded it and I can't install it, what's wrong?

    If you did follow the download instructions, maybe your phone is not compatible, go to "Settings > About Phone" and check your Android version number is 2.3.3 or up.

  • Q2: I can't find a card, what's wrong!?

    Maybe the card is not there or it's called a different way in this translation. If you're sure the card is not there and it should, contact me.

  • Q3: I found an error in a translation, how can I report it?

    In the future you will be able to report wrong translations from the app but for the time being, just write to me.

  • Q4: What other features are you planning?

    See the What's Next section for my plan for the final version.

Not So Frequently Asked

  • Q5: Where did you get these translations from?

    From a couple of different places, but mostly from the amazing site Heart of the Cards. Now before you start pointing out those guys don't want redistribution, I've hand picked, collected, fixed and formatted each one of the translations in the app, to the point many doesn't look anything like the originals.

  • Q6: Is there going to be an iOS version?

    It's on the plan, but right now I'm focusing on the android version. If you're an iOS developer and want to help me with it, please contact me.

  • Q7: Is there any hidden feature?

    Whenever you open a card, press the square left to the name for a couple of seconds and see what happens.

  • Q8: Is WS Collector open source?

    All the code for the app is available at Github

  • Q9: Why are all the contact links redirecting me to Gmail!?

    Everyone has a gmail account, but if you don't, you can contact me to wscollector at gmail dot com.

What's Next

These are a list of the features I'm planning to add in the future. Some of them are part of the final version so stay tuned for updates. I'll modify this list according to your feedback, so please write to me.

  • Custom decks. The app will allow you to save your own card lists for easy access to your own decks. Open a card in the library and press the "+" sign to add it to a custom deck, then browse your decks in the Deck Box.
  • Filter options. Want to see all the Schwarz climax cards? how about you want to look for all the Characters with power above 5000? The filter will expand the search options.
  • More languages. Right now the app is localized in english and spanish, but the translations are all in english. In the future you will be able to read the cards in spanish.
  • Card images. There's a lot of white space in the card view right now, but that will change. Shipping all the card images with the app for offline access will be very heavy so I'm thinking on letting you add your own card images.
